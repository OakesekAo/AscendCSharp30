    Changelog

    All notable changes to AscendCSharp30 are documented here.

    [Unreleased]

    Added

    • Week overview pages (Week 1-4 curriculum summaries)
    • Complete Starter READMEs for Days 09-14 (Minimal APIs)
    • Complete Program.cs files for Days 09-14 (working Minimal API projects)
    • Complete READMEs for Days 09-14 (professional documentation)
    • Fixed mkdocs.yml YAML syntax error

    Fixed

    • MkDocs build error (YAML parsing issue at line 44)
    • Navigation structure (preserved individual day links while adding week overviews)

    [v0.2.0] - 2024-12-04

    Added

    • Days 09-14 Complete: Full Minimal API implementation
    • Day 09: Minimal API Foundations (GET/POST endpoints)
    • Day 10: DTOs & API Contracts (request/response separation)
    • Day 11: Async/Await Patterns (Task-based architecture)
    • Day 12: Error Handling (status codes, validation, middleware)
    • Day 13: JSON & File I/O (export, import, search)
    • Day 14: Week 2 Capstone (complete API with analytics)
    • Comprehensive Starter READMEs for all Week 2 days
    • Professional Complete READMEs explaining working code

    Details

    • Each day's Complete example builds progressively on the previous
    • All code uses production patterns (DI, DTOs, async, error handling)
    • All endpoints documented and testable with curl/Postman

    [v0.1.0] - 2024-12-01

    Added

    • Days 01-07 Complete: Console applications
    • Day 01-03: Core C# fundamentals
    • Day 04-06: Collections and LINQ
    • Day 07: Week 1 Capstone (Job Scheduler)
    • Days 08 Complete: Dependency Injection foundation
    • Comprehensive documentation for all days
    • ServiceHub domain context throughout curriculum

    Features

    • Progressive learning path (each day builds on previous)
    • Real-world ServiceHub example throughout
    • Both Starter (learning guide) and Complete (working code) for each day
    • Professional code patterns and best practices

    Versioning

    • v0.1.0: Days 01-08 (Fundamentals + DI foundation)
    • v0.2.0: Days 09-14 (Minimal APIs, complete Week 2)
    • v0.3.0: Days 15-21 (Database, EF Core) — in progress
    • v1.0.0: Days 22-30 (Full-stack, Blazor UI, deployment) — planned

    How to Contribute

    2. Verify all links work
    3. Submit issues for broken content
    4. Suggest improvements
